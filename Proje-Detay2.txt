SEÇENEK A: CRAWL4AI (Birincil Tavsiye)
---------------------------------------
Crawl4AI açık kaynaklı bir AI-native crawler'dır.
Playwright tabanlıdır, gerçek browser fingerprint üretir.

Kurulum:
  pip install crawl4ai
  crawl4ai-setup   # Playwright browser'ları indirir

Temel kullanım:
  from crawl4ai import AsyncWebCrawler
  from crawl4ai.extraction_strategy import JsonCssExtractionStrategy
  import asyncio, json

  schema = {
    "name": "Vatan Ürün Listesi",
    "baseSelector": ".product-list--list-page",
    "fields": [
      {"name": "name",      "selector": ".product-list__product-name",  "type": "text"},
      {"name": "sku",       "selector": ".product-list__product-code",   "type": "text"},
      {"name": "price",     "selector": ".product-list__price",          "type": "text"},
      {"name": "old_price", "selector": ".product-list__current-price",  "type": "text"},
      {"name": "url",       "selector": "a.product-list-link",           "type": "attribute", "attribute": "href"}
    ]
  }

  async def scrape(url):
    async with AsyncWebCrawler(
      headless=True,
      browser_type="chromium",
      verbose=False
    ) as crawler:
      result = await crawler.arun(
        url=url,
        extraction_strategy=JsonCssExtractionStrategy(schema),
        # Proxy entegrasyonu:
        proxy="http://user:pass@proxy-host:port",
        # Browser fingerprint gizleme:
        magic=True,        # Otomatik anti-detect
        simulate_user=True # Mouse hareketi simüle eder
      )
      return json.loads(result.extracted_content)

Crawl4AI'ın sağladığı anti-detect özellikleri:
  - magic=True → navigator.webdriver = false
  - simulate_user=True → rastgele mouse/scroll hareketi
  - Gerçek browser (Chromium/Firefox/WebKit) kullanır
  - Her oturumda farklı browser fingerprint

SEÇENEK B: FİRECRAWL (Managed Servis — En Kolay)
--------------------------------------------------
Firecrawl bir SaaS scraping servisidir. İstek sizden 
değil Firecrawl'ın dağıtık altyapısından çıkar.
Kendi IP'niz hiç görünmez.

  pip install firecrawl-py

  from firecrawl import FirecrawlApp

  app = FirecrawlApp(api_key="fc-YOUR_KEY")

  # Tek ürün fiyatı:
  result = app.scrape_url(
    "https://www.vatanbilgisayar.com/samsung-galaxy-s25-fe-akilli-telefon.html",
    params={
      "formats": ["extract"],
      "extract": {
        "schema": {
          "price": "number",
          "name": "string", 
          "in_stock": "boolean"
        }
      }
    }
  )

  # Kategori tarama (crawl):
  crawl_result = app.crawl_url(
    "https://www.vatanbilgisayar.com/firsat-urunler",
    params={
      "limit": 200,
      "scrapeOptions": {"formats": ["html", "markdown"]}
    }
  )

Avantaj: Tamamen managed, IP sorununu Firecrawl halleder.
Dezavantaj: Ücretli (free tier var: 500 kredi/ay).
Fiyat: ~$16/ay başlıyor (500k kredi).

SEÇENEK C: CLOUDFLARE WORKERS (Serverless — IP'siz Mimari)
-----------------------------------------------------------
Cloudflare Workers, 200+ edge konumundan çalışır.
Her istek farklı Cloudflare datacenter IP'sinden çıkar.

Worker kodu (JavaScript — Cloudflare'de deploy edilir):
  
  // wrangler.toml:
  // name = "vatan-scraper"
  // compatibility_date = "2024-01-01"
  
  export default {
    async fetch(request, env) {
      const { url } = await request.json();
      
      const response = await fetch(url, {
        headers: {
          "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36",
          "Accept-Language": "tr-TR,tr;q=0.9",
          "Accept": "text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8",
          "Referer": "https://www.vatanbilgisayar.com/",
          "Cache-Control": "no-cache"
        }
      });
      
      const html = await response.text();
      return new Response(JSON.stringify({ html, status: response.status }), {
        headers: { "Content-Type": "application/json" }
      });
    }
  };

Python tarafında Worker'ı çağırmak:
  import requests

  def fetch_via_worker(url):
    worker_url = "https://vatan-scraper.YOUR-SUBDOMAIN.workers.dev"
    resp = requests.post(worker_url, json={"url": url})
    return resp.json()["html"]

Avantaj: Cloudflare'nin kendi IP'leri ile Cloudflare'yi bypass eder!
         (Cloudflare, kendi Worker IP'lerine güvenir.)
Free tier: 100.000 istek/gün ücretsiz.

SEÇENEK E: PLAYWRIGHT + PROXY ROTASYON (Self-Hosted)
-----------------------------------------------------
Kendi altyapında çalışır ama proxy havuzu gerektirir.

  pip install playwright
  playwright install chromium

  from playwright.async_api import async_playwright
  import asyncio, random

  PROXIES = [
    "http://user:pass@proxy1:port",
    "http://user:pass@proxy2:port",
    # ... en az 10-20 farklı proxy
  ]

  async def scrape_product(url):
    proxy = random.choice(PROXIES)
    
    async with async_playwright() as p:
      browser = await p.chromium.launch(
        headless=True,
        proxy={"server": proxy}
      )
      
      # Fingerprint maskeleme:
      context = await browser.new_context(
        viewport={"width": random.randint(1200, 1920), 
                  "height": random.randint(768, 1080)},
        user_agent="Mozilla/5.0 (Windows NT 10.0; Win64; x64) ...",
        locale="tr-TR",
        timezone_id="Europe/Istanbul",
        geolocation={"latitude": 41.0082, "longitude": 28.9784},  # İstanbul
        permissions=["geolocation"],
        extra_http_headers={
          "Accept-Language": "tr-TR,tr;q=0.9,en;q=0.8"
        }
      )
      
      # Webdriver tespitini kapat:
      await context.add_init_script("""
        Object.defineProperty(navigator, 'webdriver', {get: () => undefined});
        Object.defineProperty(navigator, 'plugins', {get: () => [1,2,3,4,5]});
        window.chrome = { runtime: {} };
      """)
      
      page = await context.new_page()
      
      # İnsan gibi davran:
      await page.goto(url, wait_until="domcontentloaded")
      await asyncio.sleep(random.uniform(1.5, 3.5))
      await page.mouse.move(random.randint(100, 800), random.randint(100, 600))
      await asyncio.sleep(random.uniform(0.5, 1.5))
      
      # JSON-LD'den fiyat çek:
      price_data = await page.evaluate("""
        () => {
          const scripts = document.querySelectorAll('script[type="application/ld+json"]');
          for (let s of scripts) {
            try {
              const d = JSON.parse(s.innerText);
              if (d['@type'] === 'Product') return d;
            } catch(e) {}
          }
          return null;
        }
      """)
      
      await browser.close()
      return price_data

BÖLÜM 3: BAN YEMEMEk İÇİN KURALLAR
===============================================================

1. REQUEST THROTTLING:
   - Aynı IP'den: max 1 istek / 3 saniye
   - Ardışık istek sayısı: max 10 → sonra IP değiştir
   - Proxy rotasyon: round-robin değil, random seç
   - Gece 02:00-07:00 yoğun tarama yap (trafik düşük, 
     anomali tespiti zayıf)

2. SESSION YÖNETİMİ:
   - Her proxy için ayrı cookie jar tut
   - Cookie'leri session boyunca koru
   - Çok sık yeni session açma (şüpheli görünür)

3. REQUEST SIRASI (insan davranışı taklit):
   Ana sayfa → Kategori → Ürün sayfası
   (Direkt ürün sayfasına git değil!)
   
   Kod:
     await page.goto("https://www.vatanbilgisayar.com/")
     await sleep(2)
     await page.goto(kategori_url)
     await sleep(1.5)
     await page.goto(urun_url)

4. HEADER'LAR (zorunlu):
   {
     "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) 
                    AppleWebKit/537.36 (KHTML, like Gecko) 
                    Chrome/122.0.0.0 Safari/537.36",
     "Accept": "text/html,application/xhtml+xml,...",
     "Accept-Language": "tr-TR,tr;q=0.9",
     "Accept-Encoding": "gzip, deflate, br",
     "Connection": "keep-alive",
     "Upgrade-Insecure-Requests": "1",
     "Sec-Fetch-Dest": "document",
     "Sec-Fetch-Mode": "navigate",
     "Sec-Fetch-Site": "same-origin",
     "Sec-Ch-Ua": '"Chromium";v="122", ...',
     "Sec-Ch-Ua-Mobile": "?0",
     "Sec-Ch-Ua-Platform": '"Windows"'
   }

5. CLOUDFLARE CHALLENGE GELİRSE:
   - 503 veya "Just a moment..." → 60-120 saniye bekle
   - Aynı proxy ile tekrar deneme! Yeni proxy al
   - Günlük ban: 24 saat o proxy'yi kara listeye al

6. RATE LIMIT ALGILAMASI:
   - HTTP 429 → exponential backoff: 1→2→4→8→16 dakika
   - HTTP 403 → proxy değiştir + 5 dakika bekle
   - HTTP 503 → Cloudflare → Playwright ile dene

## Kısa Özet: Hangi Aracı Ne Zaman Kullan?

| Araç | Kullanım Durumu | Maliyet | Zorluk |
|---|---|---|---|
| **Cloudflare Worker** | Hızlı fiyat doğrulama | Ücretsiz | Kolay |
| **GitHub Actions** | Periyodik tarama | Ücretsiz | Kolay |
| **Firecrawl** | Bulk/güvenilir tarama | $16/ay | Çok Kolay |
| **Crawl4AI** | Self-hosted, tam kontrol | Proxy maliyeti | Orta |
| **Playwright + Proxy** | Karmaşık JS siteler | Proxy maliyeti | Zor |

En pratik başlangıç kombinasyonu: **GitHub Actions** (ücretsiz zamanlamacı) + **Firecrawl** (ücretsiz tier) + **Cloudflare Worker** (ücretsiz IP maskeleme) → Sıfır sunucu maliyetiyle tam ban koruması.
